This program matches users preferences with recommended restaurants, and returns information about restaurants connectivity and the shortest paths between restaurants in the network. Users are able to type in the type of restaurants and locations they are looking for in the form, and choose one of the graph traversals (bfs/ dfs) and shortest path algorithms (bellman ford/ dijkstra) to get the similar restaurants and shortest paths between two restaurants.

I. Intructions

Required modules
  • json
  • urllib3
  • requests
  • flask
  • geopy.distance
  • heapq
API key

Your private API Key will be automatically generated after you create your app on Yelp developer. For detailed instructions, refer to Yelp’s authentication guide.

Run the program

Run python3 checkpoint_advanced.py on terminal

II. Interaction Options

Basic Search

Type in the type of restaurants and locations you are looking for in the form. The program will return a list of restaurants relevant to your preference within the chosen area and render the result on the website.

Advanced Search

This advanced search is based on the graph database from your basic search result.

1. Graph traversal

After basic search, choose one restaurant from your basic search result and choose a graph traversal algorithm, BFS or DFS, the program will return a list of restaurants that are connected to your chosen restaurant.

2. Shortest Path

After basic search, choose two restaurants (from A restaurant to B restaurant) from your basic search result and choose a shortest path algorithm, Bellman-Ford or Dijkstra’s. The program will return the shortest path from A restaurant to B restaurant.
